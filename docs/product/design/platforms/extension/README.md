# Chrome Extension Design

Platform-specific design documentation for the Chrome extension.

## Folders

- **screens/** - High-fidelity screen designs and mockups
- **wireframes/** - Low-fidelity wireframes and user flows
- **specs/** - Detailed design specifications and measurements

## Platform Constraints

- Maximum popup width: 800px
- Maximum popup height: 600px
- Must follow Chrome extension design guidelines
- Consider keyboard shortcuts and browser integration
