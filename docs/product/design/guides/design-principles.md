## How to Use This Guide

**When making design decisions:**

1. Check if decision aligns with principles
2. If principles conflict, refer to Core Values priority
3. Document exceptions in Learnings

**When evaluating designs:**

- Does it match Brand Personality?
- Does it achieve Target Emotion?
- Does it reinforce Differentiator?

**Good principles are:**

- Opinionated (help make tradeoff decisions)
- Memorable (3-5 max)
- Actionable (can evaluate designs against them)

## When to Revisit

- After shipping 2-3 features (initial definition)
- When team makes inconsistent decisions
- When explaining same rationale repeatedly
- When product direction shifts
