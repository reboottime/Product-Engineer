## Instructions for Using This Template

**When to create competitive analysis:**
- Entering new market or feature category
- Annual market review
- Major competitor launches
- Pricing/positioning decisions
- Feature prioritization

**Tips:**
1. **Focus on differentiation** - Don't just list features, explain why ours is better
2. **Include evidence** - Screenshots, user reviews, actual usage
3. **Update regularly** - Competitive landscape changes, set review cadence
4. **Connect to strategy** - Every insight should inform a decision or requirement
5. **Be honest about weaknesses** - Helps us prioritize improvements
6. **Topic-based naming** - `meal-tracking-analysis.md` or `myfitnesspal.md`
7. **Update in-place** - Edit existing analysis, track updates in YAML frontmatter
8. **Delete this section** when creating actual competitive analysis
